## System Overview

The Financial Advisor is a multi-agent AI system that provides comprehensive financial advice through a structured process:

1. **Market Data Analysis** - Gathers and analyzes market information
2. **Trading Strategy Development** - Creates tailored trading strategies
3. **Execution Planning** - Develops detailed execution plans
4. **Risk Assessment** - Evaluates risks and provides mitigation strategies

## Key Features

- **Multi-Agent Architecture**: Specialized agents for different aspects of financial advice
- **Real-time Data**: Integration with Google Search for current market information
- **Status Logging**: Comprehensive tracking of agent execution and performance
- **Cloud-Native**: Built for Google Cloud Platform with Vertex AI
- **Scalable**: Designed for horizontal scaling and high availability

## Technology Stack

- **Google ADK**: Agent Development Kit for multi-agent orchestration
- **Vertex AI**: Managed AI platform for agent execution
- **Gemini 2.5 Pro**: Large language model for agent reasoning
- **CloudSQL PostgreSQL**: Database for status logging and analytics
- **Google Cloud Platform**: Infrastructure and hosting
